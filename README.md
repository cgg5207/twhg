# TWHG(Trello Webhook Github)

## Overview

When a user creates a pull request with the trello card link, automatically move the card from the "doing" to the "reviewing" list using the GitHub webhook and Trello API.

## Installation

*Requirement*

* Ruby2+
* MySQL
* Redis
